## 반드시 따라야 할 7가지 디자인 원칙
### 원칙 1: 섹션 배경 리듬
홈페이지가 단조롭지 않은 이유는 섹션마다 레이아웃이 다르기 때문이다.

히어로: 풀 너비, 텍스트 중심, 어두운 배경
트렌딩: 리스트 형태, 번호 매김, 밝은 배경
컬렉션: 대표 카드(크게) + 나머지(작게), 2컬럼
최신: 가로형 카드 리스트, 세로 스택
하단 CTA: 다시 어두운 배경, 텍스트 중심

이 "밝음-밝음-밝음-어두움" 리듬이 스크롤하면서 시각적 변화를 만든다.
KNS도 이 리듬을 따르라. 모든 섹션이 흰 배경 + 카드 나열이면 죽은 페이지가 된다.

### 원칙 2: 계층 구조가 명확한 타이포그래피

영문 소제목 (작고 얇음, 12~14px, 대문자, 회색 또는 악센트)
한글 섹션 제목 (크고 굵음, 24~28px, 검정)
카드 제목 (중간, 18~20px, bold)
요약/날짜 (작음, 14~16px, 회색)
이 4단계가 일관되게 유지됨.

### 원칙 3: 카드 안의 정보 순서
모든 카드에서 정보가 같은 순서로 나옴:

썸네일 이미지
카테고리 태그 (컬러 라벨) + 날짜
(선택) "추천 콘텐츠" 라벨
제목 (가장 크고 굵음)
요약 (1~3줄)
작성자 정보 (아바타 + 이름)

이 순서가 바뀌지 않는다. 어떤 카드든 눈이 같은 패턴으로 읽힌다.

### 원칙 4: 컬러 사용의 절제
전체적으로 거의 흑백+중성 톤이다.

배경: 흰색 또는 매우 밝은 회색
텍스트: 검정 또는 짙은 회색
컬러가 쓰이는 곳: 카테고리 태그 라벨 (파랑, 초록 등 소형), CTA 버튼, 메달 이모지
과도한 컬러 없이도 "정돈되어 있다"는 느낌을 줌

### 원칙 5: 여백이 콘텐츠만큼 중요

섹션과 섹션 사이: 넉넉한 간격 (64px 이상)
카드와 카드 사이: 일정한 간격
카드 내부: 텍스트가 빽빽하지 않음
히어로 영역: 텍스트 주변에 큰 여백으로 시선 집중

### 원칙 6: "전체보기" 패턴
각 컬렉션 섹션의 우상단에 항상 "전체보기 →" 링크가 있다.
이것은 사용자에게 "여기 더 있다"는 신호를 주면서, 홈페이지를 깔끔하게 유지한다.
KNS도 각 섹션에 이 패턴을 적용하라.

### 원칙 7: 페이지네이션 > 무한스크롤
최신 콘텐츠 영역은 명시적 페이지 전환을 사용한다.
"이전 / 다음" + 현재 페이지 표시.
무한스크롤을 쓰지 않는다.

### KNS 매거진 홈 레이아웃 최종 명세
위 레퍼런스를 KNS에 맞게 재해석한 최종 홈 구조:
[상단 띠 배너 — 시즌 CTA 또는 숨김]
[헤더 — 로고 + 학년별 + 주제별 + 검색 + 상담신청]

[히어로 — 네이비 배경, KNS 브랜드 메시지 + 학년 빠른 진입 또는 히어로 콘텐츠]

[지금 많이 보는 콘텐츠 — 순위 리스트 3~5개, 아이보리 배경]

[주제 컬렉션 A — 대표 카드(크게) + 나머지(작게), 흰색 배경]

[설명회·이벤트 배너 — 운영자 등록 배너 1~2개, 또는 숨김]

[주제 컬렉션 B — 같은 패턴, 아이보리 배경으로 구분]
