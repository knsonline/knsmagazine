# 마케팅팀 실사용 체크리스트

## 오픈 전 하루 전

1. 관리자 로그인 계정이 정상인지 확인
2. 홈 대표 콘텐츠가 지정되어 있는지 확인
3. 학년별 진입에서 빈 화면이 아닌지 확인
4. 최신 콘텐츠 3건 이상이 공개 상태인지 확인
5. CTA 링크가 실제 상담/설명회 링크로 연결되는지 확인
6. 배너가 있다면 시작일/종료일과 링크를 다시 확인
7. 모바일에서 홈과 상세 화면을 한 번씩 확인

## 오픈 당일

1. `/api/health` 접속 후 `status: ok` 확인
2. 홈 `/` 접속 확인
3. 관리자 `/admin/login` 접속 확인
4. 콘텐츠 1건 상세 진입 확인
5. CTA 클릭 링크 확인
6. 배너 클릭 링크 확인
7. UptimeRobot이 정상 감시 중인지 확인

## 콘텐츠 등록 시 체크

1. 제목이 학부모 기준으로 이해 쉬운지 확인
2. 요약이 1~2문장으로 정리됐는지 확인
3. 학년/주제/타입이 맞는지 확인
4. 썸네일이 업로드 후 깨지지 않는지 확인
5. 공개 여부를 다시 확인
6. 필요하면 CTA 연결을 선택

## 배너 등록 시 체크

1. 이미지가 가로형 배너 비율에 맞는지 확인
2. 링크 URL이 실제 신청/안내 페이지인지 확인
3. 노출 기간이 맞는지 확인
4. 활성 상태인지 확인
5. 홈에서 배너 이미지가 잘 보이는지 확인

## 주 1회 운영 점검

1. `/api/health`
2. 홈 배너 노출
3. 최신 콘텐츠 업데이트 여부
4. 종료된 배너가 남아 있지 않은지 확인
5. 상담/설명회 링크가 만료되지 않았는지 확인

## 문제가 보이면 바로 확인할 것

1. health endpoint가 `ok`인지
2. 이미지가 아예 안 보이면 최근 업로드한 파일인지
3. 링크가 틀리면 관리자에서 URL 수정 가능한지
4. 관리자 로그인 자체가 안 되면 개발 담당자에게 바로 전달
